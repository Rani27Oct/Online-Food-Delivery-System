# 🍔 Online Food Delivery System

## 📌 Project Description
Online Food Delivery System is a responsive web application developed using HTML, CSS, and JavaScript. Users can browse food items, search foods, view ratings, place orders, and enter delivery details.

## 🚀 Features
- Food Menu Display
- Search Food Functionality
- Food Ratings
- Order Now Button
- Delivery Address Form
- Special Offers Section
- Contact Information
- Responsive Design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages
