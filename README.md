# Online Food Delivery System

## Project Overview
The Online Food Delivery System is a full stack web application developed using Java, Spring Boot, Hibernate, and MySQL. This application allows users to browse food items, place orders, and manage bookings online.

## Features
- User Registration and Login
- Browse Food Menu
- Add to Cart
- Place Orders
- Admin Dashboard

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Java
- Spring Boot
- Hibernate

### Database
- MySQL

1. Clone the repository
2. Open in VS Code or Eclipse
3. Configure MySQL database
4. Run Spring Boot application
